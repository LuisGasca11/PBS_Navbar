# PRUEBA PROGRAMADOR - black_sheep

Este proyecto es una página web inspirada en el sitio oficial de [World of Warcraft](https://worldofwarcraft.blizzard.com/es-es/), 
con el objetivo de replicar el **navbar** y el **footer**.

Además, incluye un **sistema de login** que permite cambiar el fondo del `body` y la imagen principal del navbar.

## Funcionalidades
- **Login:** cambia dinámicamente el fondo y la imagen principal del sitio.  
- **Drag & Drop:** disponible únicamente al iniciar sesión.  
  - En la versión **de escritorio**, permite reorganizar los elementos del navbar.  
  - En la versión **móvil**, permite reorganizar los elementos del menú.

---

## Tecnologías utilizadas
- React + Vite
- Tailwind CSS
- Lucide React (iconos)
